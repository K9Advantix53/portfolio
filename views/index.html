    SYNAPSE

    Site || Github

    Synapse is a web application that connects developers with businesses. Users will be able to both create new projects and join existing projects as a team member. Teams will have access to chatroom for logistics planning. This web app features a React front end with a Rails backend and PostgreSQL database. Search engine made in React using RESTful APIs.

    Google Maps API and geocoder to generate dynamic maps, ActionCable to set up live chats.

    Created in a 2-week sprint for the final project of Launch Academy.

    DnD Planner (In Progress)

    Site || Github

    DnD Planner is a web application that serves as a Dungeons and Dragons (5th Ed) encyclopedia as well as a character tracker. Users can look up spells, equipments, monsters, as well as the 12 starting character classes. DnD Planner is an exploration of external APIs, React.js conditional loaders, and a study of the React.js lifecycle.

    Uses the D&D 5e API.

    Created due to an excess of page turning during game nights.
