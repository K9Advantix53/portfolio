SOLO PROJECTS

SYNAPSE

Site || Github

Synapse is a web application that will serve to connect developers with businesses. Users will be able to both create new projects and join existing projects as a developer. This web app features a React front end with a Rails backend and Postgres database. I also used the Google Maps API to generate dynamic maps, implemented ActionCable to set up live chats and made a custom search engine in React using RESTful APIs.

Created in a 2-week sprint for the final project of Launch Academy.

DnD Planner (In Progress)

Site || Github

DnD Planner is a web application that serves as an Dungeons and Dragons (5th Ed) encyclopedia and well as a character tracker. Users can look up spells, equipments, monsters, as well as the 12 starting character classes. DnD Planner is an exploration of external APIs, React.js conditional loaders, and a study of the React.js lifecycle.

Created due to an excess of page turning during game nights.
