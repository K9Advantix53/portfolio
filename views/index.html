SYNAPSE

synapse-coder.herokuapp.com

back-end focus

Synapse is a web application that will serve to connect developers with businesses. Users will be able to both create new projects and join existing projects as a developer. This web app features a React front end with a Rails backend and Postgres database. I also used the Google Maps API to generate dynamic maps, implemented ActionCable to set up live chats and made a React search engine.
